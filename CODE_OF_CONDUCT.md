# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone, regardless of background or experience.

This project is intended to support research, education, and software development in an open and collaborative manner.

## Expected Behavior

Participants are expected to:

- Be respectful and professional.
- Welcome constructive feedback.
- Assume good intentions.
- Help create a positive learning environment.
- Focus discussions on improving the project.

## Unacceptable Behavior

Examples include:

- Harassment or discrimination.
- Personal attacks.
- Trolling or inflammatory comments.
- Publishing private information without permission.
- Disruptive or abusive behavior.

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation
- Community interactions related to this repository

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for behavior that violates this Code of Conduct.

## Reporting

If you encounter behavior that violates this Code of Conduct, please open a private communication with the project maintainers.

Reports will be handled respectfully and confidentially whenever possible.
